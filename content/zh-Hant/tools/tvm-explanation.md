---

## 現值（PV）

**現值（PV）**是未來一筆金額或一系列支付在特定收益率下的當前價值。

PV回答的問題是：*"為了達到未來目標,我今天需要投資多少？"*或*"我未來支付的當前價值是多少？"*

僅計算有終值的PV公式是：

$$
PV = \frac{FV}{(1 + r)^n}
$$

如果涉及定期支付（PMT）,公式擴展為：

$$
PV = \frac{FV}{(1 + r)^n} + PMT \times \frac{(1+r)^n - 1}{r(1+r)^n}
$$

其中：
- PV = 現值
- FV = 終值
- PMT = 每期支付額
- r = 每期利率
- n = 期數

## 終值（FV）

**終值（FV）**表示資產或現金在未來特定日期的價值,基於假定的增長率。

FV回答的問題是：*"我的投資在未來會值多少錢？"*或*"我的定期支付在未來的價值會是多少？"*

僅有初始投資的FV計算公式是：

$$
FV = PV \times (1 + r)^n
$$

如果涉及定期支付（PMT）,公式擴展為：

$$
FV = PV \times (1 + r)^n + PMT \times \frac{(1+r)^n - 1}{r}
$$

其中：
- FV = 終值
- PV = 現值
- PMT = 每期支付額
- r = 每期利率
- n = 期數

## 支付額（PMT）

**支付額（PMT）**是年金中的定期支付金額。

PMT回答的問題是：*"為了償還貸款,我每期需要支付多少？"*或*"為了達到目標,我應該定期儲蓄多少？"*

用現值（如貸款金額）計算PMT的公式是：

$$
PMT = \frac{PV \times r \times (1 + r)^n}{(1 + r)^n - 1}
$$

如果還有終值目標,公式擴展為：

$$
PMT = \frac{PV \times r \times (1 + r)^n - FV \times r}{(1 + r)^n - 1}
$$

其中：
- PMT = 每期支付額
- PV = 現值
- FV = 終值
- r = 每期利率
- n = 期數

## 期數（N）

**期數（N）**代表投資或貸款中的複利計算期總數。

N回答的問題是：*"達到我的財務目標需要多長時間？"*或*"償還貸款需要多少次付款？"*

沒有定期支付時計算N的公式是：

$$
N = \frac{\ln(FV / PV)}{\ln(1 + r)}
$$

當涉及支付時,計算變得更複雜,通常需要數值方法求解：

$$
PV(1+r)^N + PMT\frac{(1+r)^N-1}{r} = FV
$$

其中：
- N = 期數
- FV = 終值
- PV = 現值
- PMT = 每期支付額
- r = 每期利率
- ln = 自然對數

## 利率（I/Y）

**利率（I/Y）**是投資或貸款的年利率,以百分比表示。

I/Y回答的問題是：*"我的投資獲得了什麼回報率？"*或*"我的貸款利率是多少？"*

計算I/Y涉及找到滿足以下方程的利率r：

$$
PV(1+r)^N + PMT\frac{(1+r)^N-1}{r} = FV
$$

當涉及PMT時,該方程通常需要迭代方法來求解r。

其中：
- I/Y = 年利率
- FV = 終值
- PV = 現值
- PMT = 每期支付額
- N = 期數

## 複利頻率

複利頻率影響利息隨時間的積累方式。更頻繁的複利計算通常會導致更高的有效利率。上述計算假設支付頻率與複利頻率相匹配,但當這些頻率不同時,公式可以進行調整。
