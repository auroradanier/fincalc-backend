# Utilisation du Mode Amortissement de la Calculatrice BAII Plus

[CFA]{.p-1 .text-xs .rounded-md .bg-gray-200} [FRM]{.p-1 .text-xs .rounded-md .bg-gray-200}

## I. Introduction

Le mode amortissement de la calculatrice BAII Plus est un outil puissant pour l'analyse des prêts et des hypothèques. Il permet l'examen détaillé des calendriers de paiement, y compris la répartition entre le principal et les intérêts.

## II. Principes de Base de l'Amortissement

Le mode amortissement affiche :
- Les paiements d'intérêts pour une période donnée
- Les montants du principal remboursé pour une période donnée
- Le solde restant après des paiements spécifiques
- Les paiements totaux sur plusieurs périodes

## III. Accéder au Mode Amortissement

1. Saisissez d'abord les variables TVM :
   - N (nombre total de paiements)
   - I/Y (taux d'intérêt annuel)
   - PV (montant du prêt)
   - PMT (paiement régulier)
   - FV (valeur future, généralement 0)

2. Appuyez sur [2nd]{.p-1 .rounded-md .bg-gray-200} + [0]{.p-1 .rounded-md .bg-gray-200} (AMORT)

## IV. Exemples Pratiques

**Exemple 1 : Analyse d'un Prêt Hypothécaire**

> Analysez un prêt hypothécaire avec les conditions suivantes :
> - Montant du prêt : 200 000 €
> - Durée : 30 ans
> - Taux d'intérêt annuel : 4,5 %
> - Paiements mensuels

1. Saisissez les variables TVM :
   - N = 360 (30 ans × 12 mois)
   - I/Y = 4,5
   - PV = 200000
   - PMT (à calculer)
   - FV = 0
   - P/Y = 12 (paiements mensuels)
   - C/Y = 12 (capitalisation mensuelle)

2. Appuyez sur [PMT]{.p-1 .rounded-md .bg-gray-200} pour le paiement mensuel

3. Appuyez sur [2nd]{.p-1 .rounded-md .bg-gray-200} + [0]{.p-1 .rounded-md .bg-gray-200} pour le mode amortissement

4. Saisissez les paiements à analyser (par exemple, 1-12 pour la première année)

**Exemple 2 : Analyse de Périodes de Paiement Spécifiques**

> Examinez les paiements 13-24 (deuxième année) :

1. Dans le mode amortissement :
   - Saisissez 13 [ENTER]{.p-1 .rounded-md .bg-gray-200}
   - Saisissez 24 [ENTER]{.p-1 .rounded-md .bg-gray-200}
   - Appuyez sur [↓]{.p-1 .rounded-md .bg-gray-200} pour voir :
     - BAL (solde restant)
     - PRN (principal remboursé)
     - INT (intérêts payés)

**Exemple 3 : Paiements Cumulés**

> Calculez les paiements totaux pour les cinq premières années :

1. Dans le mode amortissement :
   - Saisissez 1 [ENTER]{.p-1 .rounded-md .bg-gray-200}
   - Saisissez 60 [ENTER]{.p-1 .rounded-md .bg-gray-200}
   - L'affichage montre :
     - Principal total remboursé
     - Intérêts totaux payés
     - Solde après 5 ans

## V. Conseils et Dépannage

- **Paramètres de Paiement :** Assurez-vous que P/Y et C/Y sont correctement définis
- **Convention de Signe :** Veillez à utiliser correctement les nombres positifs et négatifs
- **Vérification des Données :** Utilisez les touches [↑]{.p-1 .rounded-md .bg-gray-200}/[↓]{.p-1 .rounded-md .bg-gray-200} pour vérifier toutes les valeurs

## VI. Fonctions Avancées

**Analyse des Paiements Partiels**
- Analysez des paiements individuels dans une période
- Comparez différentes périodes
- Calculez l'impact des paiements supplémentaires

**Calcul du Solde Restant**
- Déterminez le solde à tout moment
- Calculez l'impact du remboursement anticipé
- Analysez différents scénarios d'amortissement
