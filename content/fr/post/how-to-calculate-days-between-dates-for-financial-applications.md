# Comment calculer les jours entre les dates pour les applications financières

[CFA]{.p-1 .text-xs .rounded-md .bg-gray-200} [FRM]{.p-1 .text-xs .rounded-md .bg-gray-200} 

Le mode date de la calculatrice financière est un outil puissant pour les professionnels de la finance qui ont besoin de calculer les jours entre les dates ou de déterminer les dates futures/passées en fonction d'une date connue et d'un nombre de jours. Ce tutoriel vous guidera dans l'utilisation efficace du mode date, y compris la compréhension de la différence entre les méthodes de calcul Réel et 360 jours.

## 1. Comprendre les bases du mode Date :

Le mode date dans la calculatrice financière a quatre composants principaux :

**DT1:** La première date de votre calcul.

**DT2:** La deuxième date de votre calcul.

**DBD:** Jours entre les dates - le nombre de jours entre DT1 et DT2.

**DAY:** La méthode de calcul - soit Réel (année de 365 jours) soit 360 (mois de 30 jours).

## 2. Accéder au mode Date :

**Ouvrir la calculatrice financière :**
Commençons par ouvrir la [Calculatrice financière en ligne](https://www.baiiplus.com/)

Appuyez sur [2nd]{.p-1 .rounded-md .bg-gray-200} puis sur [DATE]{.p-1 .rounded-md .bg-gray-200} (la fonction DATE est au-dessus de la touche 1). Cela entrera en mode Date. Vous verrez "DT1=" affiché, ainsi que la valeur actuelle pour DT1.

![Interface du mode Date](/images/calculator_date_mode.png){.mx-auto}

## 3. Naviguer entre DT1, DT2, DBD et DAY :

Utilisez les touches flèche haut (▲) et flèche bas (▼) pour vous déplacer entre les quatre variables :

- Appuyez sur [▼]{.p-1 .rounded-md .bg-gray-200} une fois depuis DT1 pour passer à DT2
- Appuyez à nouveau sur [▼]{.p-1 .rounded-md .bg-gray-200} pour passer à DBD
- Appuyez encore sur [▼]{.p-1 .rounded-md .bg-gray-200} pour passer à DAY
- Utilisez [▲]{.p-1 .rounded-md .bg-gray-200} pour vous déplacer dans la direction inverse

## 4. Entrer les dates :

Lorsque DT1= ou DT2= est affiché :

1. Saisissez la date au format MM-JJ-AAAA
2. Appuyez sur [ENTRER]{.p-1 .rounded-md .bg-gray-200} pour confirmer

**Exemple :** Pour entrer le 15 janvier 2024 au format US, tapez : `01-15-2024` et appuyez sur [ENTRER]{.p-1 .rounded-md .bg-gray-200}

## 5. Calculer les jours entre les dates (DBD) :

Après avoir entré DT1 et DT2 :

1. Naviguez jusqu'à DBD en utilisant les touches fléchées
2. Appuyez sur [CPT]{.p-1 .rounded-md .bg-gray-200} pour calculer le nombre de jours entre les deux dates
3. Le résultat sera affiché en fonction de la méthode de calcul actuelle (Réel ou 360)

**Exemple :** Si DT1 est le 01-15-2024 et DT2 est le 02-15-2024, appuyer sur [CPT]{.p-1 .rounded-md .bg-gray-200} lorsque DBD est affiché affichera 31 jours en mode Réel ou 30 jours en mode 360.

## 6. Comprendre les méthodes Réel vs 360 jours :

La calculatrice BAII Plus offre deux méthodes pour calculer les jours entre les dates :

**Réel :** Utilise le calendrier réel avec 365 jours par an (366 les années bissextiles). C'est la méthode de calendrier standard.

**360 :** Utilise un calendrier financier simplifié avec 30 jours par mois et 360 jours par an. Cette méthode est couramment utilisée dans certains calculs financiers, en particulier dans les calculs d'obligations et d'intérêts.

Pour basculer entre ces méthodes :

1. Naviguez jusqu'à DAY en utilisant les touches fléchées
2. Appuyez sur [CPT]{.p-1 .rounded-md .bg-gray-200} pour basculer entre "Réel" et "360"

Alternativement, lorsque vous êtes sur l'écran DAY, vous pouvez appuyer sur [2nd]{.p-1 .rounded-md .bg-gray-200} + [ENTRER]{.p-1 .rounded-md .bg-gray-200} pour basculer entre les méthodes.

## 7. Calculer les dates futures ou passées :

En mode Réel, vous pouvez calculer une date future ou passée en fonction d'une date connue et d'un nombre de jours :

**Pour calculer DT1 (date passée) :**
1. Entrez DT2 (la date connue)
2. Entrez DBD (le nombre de jours)
3. Naviguez jusqu'à DT1
4. Appuyez sur [CPT]{.p-1 .rounded-md .bg-gray-200} pour calculer la date qui se trouve DBD jours avant DT2

**Pour calculer DT2 (date future) :**
1. Entrez DT1 (la date connue)
2. Entrez DBD (le nombre de jours)
3. Naviguez jusqu'à DT2
4. Appuyez sur [CPT]{.p-1 .rounded-md .bg-gray-200} pour calculer la date qui se trouve DBD jours après DT1

**Remarque :** En mode 360, la fonction CPT ne fonctionne pas pour calculer DT1 ou DT2.

## 8. Notes importantes sur le mode 360 jours :

Lorsque vous utilisez la méthode de calcul à 360 jours :

- Chaque mois est traité comme ayant exactement 30 jours
- L'année est traitée comme ayant exactement 360 jours
- Si DT1 et DT2 sont la même date, DBD renverra 0

## 9. Applications pratiques :

Le mode date est particulièrement utile pour :

- Les calculs d'obligations où vous devez déterminer les jours entre l'émission et l'échéance
- Les calculs d'intérêts courus
- La planification de l'échéancier de projet
- Les calculs de durée de prêt
- Le reporting financier où des décomptes de jours spécifiques sont requis

En maîtrisant le mode date de votre calculatrice financière, vous pourrez effectuer des calculs précis basés sur les dates, essentiels pour de nombreuses applications financières, de la tarification des obligations aux calculs d'intérêts.
